# Проект: Место-React
Учебный одностраничный сайт с адаптивным дизайном и интерактивными элементами сделанный на React.js
Из функционала: регистрация и авторизация, маршрутизация (react-router)

##### Технологии
* html
* css: flex-box, grid, media-queries
* js (react)
